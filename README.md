========

A [Labeled Tab-separated Values (LTSV)](http://ltsv.org/) encoder for [uber-go/zap: Fast, structured, leveled logging in Go.](https://github.com/uber-go/zap).

## Installation

```
go get -u github.com/hnakamur/zap-ltsv/...
```
